# Images des cartes Services

Placez vos images ici pour remplacer les images par défaut (Unsplash).

## Chemins des fichiers attendus

| Carte | Fichier | Dimensions recommandées |
|-------|---------|-------------------------|
| Livraison à domicile | `livraison.jpg` ou `livraison.webp` | 600×450 px (ratio 4:3) |
| Véhicules (G1, G2, G3) | `vehicules.jpg` ou `vehicules.webp` | 600×450 px (ratio 4:3) |
| Groupage maritime (220L) | `groupage.jpg` ou `groupage.webp` | 600×450 px (ratio 4:3) |

## Utilisation

Si un fichier existe à `public/images/services/{id}.jpg` (ou .webp, .png), il sera utilisé en priorité. Sinon, l'image Unsplash par défaut s'affiche.
